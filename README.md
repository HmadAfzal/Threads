# Threads Mern Stack Clone


![App SS](Frontend/public/app.png)

[Demo App Link](https://youtu.be/ymA_R7lbrWs) 👈 Click here



Feature List:

-   🌟 Tech stack: MERN + Socket.io + Chakra UI
-   🎃 Authentication & Authorization with JWT
-   📝 Create Post
-   🗑️ Delete Post
-   🖊  Edit post
-   👁  View Post
-   ❤️ Like/Unlike Post
-   💬 Comment to a Post
-   👥 Follow/Unfollow Users
-   🔍 Search User
-   🌓 Dark/Light Mode
-   📱 Completely Responsive


### Setup .env file

```js
PORT=...
MONGO_ATLAS_URI=...
JWT_SECRET=...
CLOUD_NAME=...
CLOUD_API_KEY=...
API_SECRET=...
```



### Build the app

```shell
npm run build
```

### Start the app

```shell
//Backend
npm start

//Frontend
npm run dev
```
